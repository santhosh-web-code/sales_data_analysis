#E-commerce Sales Data Analysis using SQL

# Objective

Analyze sales data to extract insights on revenue, customer behavior, and product performance.

## 🛠️ Tools Used

* SQL (MySQL/SQLite)

## 📊 Key Analysis

* Total Revenue Calculation
* Category-wise Revenue
* Top Products
* Monthly Sales Trends
* Customer Spending Analysis

## 📈 Key Insights

* Electronics category generated highest revenue
* Laptops and phones are top-selling products
* Revenue shows an increasing trend over months
* High-value customers contribute major share of revenue

## 🚀 Conclusion

This project demonstrates how SQL can be used to analyze structured data and derive business insights for decision-making.
